# Herdr keybindings quick reference

## Modifier scheme

| Level | h/l (horizontal) | j/k (vertical) | Modifier |
|---|---|---|---|
| **Pane focus** | `alt + h/l` | `alt + j/k` | `alt` |
| **Swap pane** | `alt+shift + h/l` | `alt+shift + j/k` | `alt+shift` |
| **Tabs** | `ctrl + h/l` | — | `ctrl` |
| **Workspaces** | — | `ctrl + j/k` | `ctrl` |
| **Agents** | — | `ctrl+shift + j/k` | `ctrl+shift` |

## Pane navigation

| Keys | Action |
|---|---|
| `alt + j` | Focus pane down |
| `alt + k` | Focus pane up |
| `alt + h` | Focus pane left |
| `alt + l` | Focus pane right |
| `alt+shift + j` | Move/swap pane down |
| `alt+shift + k` | Move/swap pane up |
| `alt+shift + h` | Move/swap pane left |
| `alt+shift + l` | Move/swap pane right |

## Tabs — `ctrl + h/l`

| Keys | Action |
|---|---|
| `alt + n` | New tab |
| `ctrl + h` | Previous tab |
| `ctrl + l` | Next tab |
| `ctrl + 1..9` | Switch to tab 1–9 |
| `alt+shift + t` | Rename tab |
| `alt+shift + x` | Close tab |

## Panes

| Keys | Action |
|---|---|
| `alt + v` | Split vertical |
| `alt + -` | Split horizontal |
| `alt + x` | Close pane |
| `alt + z` | Zoom (toggle fullscreen) |
| `ctrl + tab` | Cycle to next pane |
| `alt + r` | Enter resize mode (then `hjkl` to resize) |

## Workspaces — `ctrl + j/k`

| Keys | Action |
|---|---|
| `alt+shift + n` | New workspace |
| `ctrl + j` | Next workspace |
| `ctrl + k` | Previous workspace |
| `alt + w` | Workspace picker |
| `alt+shift + w` | Rename workspace |
| `alt+shift + d` | Close workspace |
| `alt + g` | Goto (jump to anything) |

## Agents — `ctrl+shift + j/k`

| Keys | Action |
|---|---|
| `ctrl+shift + j` | Next agent |
| `ctrl+shift + k` | Previous agent |
| `ctrl+alt + 1..9` | Focus agent #1–9 |

## Other

| Keys | Action |
|---|---|
| `alt + q` | Detach |
| `alt + b` | Toggle sidebar |
| `alt + [` | Enter copy mode |
